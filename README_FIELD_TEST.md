# OnStreamLLM - 사용 안내

## 실행

`OnStreamLLM.exe`를 실행합니다. Python 설치는 필요하지 않습니다.
모델과 런타임 라이브러리는 배포본에 포함되지 않으며, 앱에서 다운로드합니다.

## 첫 실행 후 설정

1. **모델** 탭에서 STT·번역 모델을 다운로드합니다.
2. 필요한 **런타임 라이브러리**(Torch, SenseVoice, llama 등)를 설치합니다.
3. **엔진 시작** 후 입력/출력/화면 감지를 사용합니다.

## OBS 오버레이

- 앱의 **설정 → 서버 / 원격** 탭에서 OBS 주소를 복사합니다.
- 기본 주소: `http://127.0.0.1:17865/overlay`
- OBS 브라우저 소스는 **실행 중인 PC에서만** 접근할 수 있습니다.

## 원격 연산 (호스트 / 클라이언트)

호스트 PC는 모든 네트워크 인터페이스(`0.0.0.0`)에서 연결을 받습니다.
접근 제어는 **패스워드**와 **클라이언트 IP 화이트리스트**로 합니다.

- 패스워드는 12자 이상으로 설정하세요.
- 외부·LAN 공유 시 **클라이언트 IP 화이트리스트 사용을 권장**합니다.
- 허용할 클라이언트 IP를 체크한 뒤 서버를 시작하세요.
- 클라이언트 PC에서는 호스트 IP·포트·패스워드를 입력하고 연결합니다.

## 네트워크 보안 주의

- 원격 연산은 **신뢰할 수 있는 LAN 또는 VPN** 안에서만 사용하세요.
- 공유기 포트포워딩으로 인터넷에 직접 공개하지 마세요. 통신은 TLS로 암호화되지 않습니다.
- 저장된 패스워드는 Windows 사용자 계정에 묶어 암호화됩니다.
- Windows 방화벽 허용 시 가능하면 **사설 네트워크**만 허용하세요.
- 원격 연산 호스트를 쓰지 않을 때는 서버를 중지하세요.

## 포트

- 연산 호스트 기본 포트: `8765`
- OBS 오버레이 포트: `17865`
